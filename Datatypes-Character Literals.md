
# Ex 1c:Variables & Expressions Operator-Program to convert temperature from Celsius to Fahrenheit

## Aim
To write a Python program that reads temperature in Celsius and converts it into Fahrenheit using the formula
Fahrenheit = (Celsius × 1.8) + 32
## Algorithm
1.Start the program.

2.Read the temperature in Celsius from the user.

3.Convert the value into Fahrenheit using:
F = (C × 1.8) + 32

4.Display the Celsius and Fahrenheit values.

5.End the program.

## Program
```
celsius = float(input())

fahrenheit = (celsius * 1.8) + 32

print(f"Celsius = {celsius:.2f}")
print(f"Fahrenheit = {fahrenheit:.2f}")
```
