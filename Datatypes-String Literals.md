# Ex 1e:Datatypes-Convert integer to float

## Aim
To write a Python program to read an integer and convert it into a float value.
## Algorithm
1.Start the program.

2.Read an integer from the user.

3.Convert the integer to float using float().

4.Display the float value.

5.End the program.

## Program
```
a=int(input())
b=float(a)
print(f"The float value of {a} is {b}")
```
